AutoCAD LISP Scripts

LISP Automations

A collection of custom AutoCAD routines to eliminate repetitive drafting tasks. Download the scripts and load them directly into your CAD environment via the APPLOAD command.

SpatialLookAlike.arx (C++ Engine)

Mesin utama berkecepatan tinggi untuk Windows. Cukup APPLOAD file ini (UI sudah tertanam) untuk Import/Export KML dan Live Map.

SLA_Master_Mac.lsp

Versi mesin LISP murni khusus untuk lingkungan macOS yang tidak mendukung format binary .arx.

SLA_Master_Win.lsp (Legacy)

Mesin LISP cadangan (fallback) untuk pengguna AutoCAD versi lama di Windows jika file .arx gagal dimuat.

BF2T.lsp

Otomatisasi pairing Block References dengan Text terdekat secara akurat.

NB.lsp

Node Bridge: Mengubah kumpulan region/garis terpisah menjadi Polygon tunggal (Unified).

RQv2.lsp

Rename Queue: Mengganti nama (rename) teks atau objek secara massal dan berurutan.

AddTextAbove4.lsp

Menambahkan teks anotasi secara presisi pada posisi tertentu di atas objek/garis.

S3-Sum_Numbers.lsp

Kalkulator otomatis: Menjumlahkan nilai angka dari sekumpulan teks yang di-select.

ShrinkPoly.lsp

Otomatisasi pengecilan dimensi (offset inward) pada objek Polygon dengan cepat.

SPMAuto.lsp

Script utilitas untuk memproses dan mengeksekusi standardisasi objek secara otomatis.

T2M_Enhanced.lsp

Konversi objek DText (teks biasa) menjadi MText (Multiline) tanpa merusak format asal.
